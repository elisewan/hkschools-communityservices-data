# Emails - Data Utilities

A collection of Python data acquisition tools for Hong Kong public data sources.

## Tools

| Tool | Purpose |
|------|---------|
| **School Email Finder** | Find contact emails for schools using Brave Search |
| **Carers Directory Scraper** | Extract organization contacts from carers.hk |
| **School Email Merger** | Merge school names with found emails using fuzzy matching |
